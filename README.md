# Eduarena

## 项目简介

当前页面主要分为两个部分：

- `Qualitative Review`：查看对话样本与人工评分结果。
- `Leaderboard`：对比模型在教学质量与通用基准上的综合表现。

项目基于 `React 19 + Vite`，适合用作小型数据展示原型、内部评审看板，或 leaderboard 可视化模板。

## 开发

```bash
npm install
npm run dev
```

如果需要同时启动前端和本地评分 API：

```bash
npm start
```

Windows 下可直接运行：

```bat
start.cmd
```

## 构建

```bash
npm run build
```
