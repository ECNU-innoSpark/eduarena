# hi_react_cc

一个用 React 构建的轻量级可视化项目，用于展示 LLM 教学能力评审结果与综合 leaderboard。项目保持极简文件结构，便于快速修改数据、迭代界面和部署静态页面。

## 项目简介

当前页面主要分为两个部分：

- `Qualitative Review`：查看对话样本与人工评分结果。
- `Leaderboard`：对比模型在教学质量与通用基准上的综合表现。

项目基于 `React 19 + Vite`，适合用作小型数据展示原型、内部评审看板，或 leaderboard 可视化模板。

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
